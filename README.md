# lex-time-ucs
Herramienta en Python para el cálculo automático de plazos procesales civiles en Perú, integrando días hábiles, fines de semana y feriados nacionales.

Este script automatiza el conteo de plazos procesales basándose en el Código Procesal Civil peruano.

# Características principales:
- Cálculo preciso: Determina la fecha de vencimiento según el tipo de recurso (Casación, Apelación, Tacha, etc.).
- Calendario Judicial: Utiliza la librería pandas para excluir automáticamente fines de semana.
- Feriados 2026: Incluye una base de datos actualizada de feriados nacionales en Perú para evitar errores en el cómputo.
- Fácil uso: Interfaz por consola sencilla para abogados y estudiantes de derecho.
